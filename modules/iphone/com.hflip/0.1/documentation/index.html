hflip Module

Description

Makes flipping easy and workable when toImage() doesn't see the flipped image

Accessing the hflip Module

To access this module from JavaScript, you would do the following:

var hflip = require("com.hflip");

The hflip variable is a reference to the Module object.

Usage

You just need an eventlistener and make sure you put it before the actual action, otherwise it won't work. See example app.js for the simplest working version
